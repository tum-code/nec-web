I. Introduction to Web Development (Chapters 1-5)

1. Introduction to Web Development

Web development refers to the building, creating, and maintaining of websites. It includes aspects such as web design, web publishing, web programming, and database management.

// Web Development involves three main layers:

II. HTML (Chapters 6-25)

6. HTML Basics

HTML (HyperText Markup Language) is the standard markup language for documents designed to be displayed in a web browser.

<!-- Basic HTML Elements -->
<h1>This is a Heading</h1>
<h2>This is a Subheading</h2>
<p>This is a paragraph.</p>
<a href="https://example.com">This is a link</a>
<img src="image.jpg" alt="Description">

7. HTML Elements

HTML elements are the building blocks of HTML pages. Each element can have attributes and content.

<!-- Common HTML Elements -->

<!-- Block-level elements -->
<div>, <p>, <h1>-<h6>, <ul>, <ol>, <li>

<!-- Inline elements -->
<span>, <a>, <strong>, <em>, <img>

<!-- Self-closing elements -->
<br>, <hr>, <img>, <input>, <meta>

8. HTML Attributes

HTML attributes provide additional information about HTML elements and are always specified in the start tag.

<!-- Common HTML Attributes -->

<a href="url" target="_blank">Link</a>
<img src="image.jpg" alt="text" width="500">
<input type="text" id="name" name="name" required>
<div class="container" id="main">Content</div>

9. HTML Headings

HTML headings are defined with the h1 to h6 tags. h1 defines the most important heading and h6 defines the least important heading.

<!-- HTML Heading Hierarchy -->

<h1>Main Title (Page Title)</h1>
<h2>Section Heading</h2>
<h3>Subsection Heading</h3>
<h4>Minor Heading</h4>
<h5>Small Heading</h5>
<h6>Smallest Heading</h6>

10. HTML Paragraphs

The HTML p element defines a paragraph. Browsers automatically add some space (margin) before and after each p element.

<!-- Paragraph Examples -->

<p>This is a paragraph. It contains text and other inline elements.</p>
<p>This is another paragraph. Paragraphs are block-level elements.</p>

<!-- Paragraph with inline elements -->
<p>
    This paragraph contains <strong>bold text</strong>,
    <em>italic text</em>, and a 
    <a href="#">link</a>.
</p>

III. CSS (Chapters 26-50)

26. Introduction to CSS

CSS (Cascading Style Sheets) is used to style and layout web pages â€” for example, to alter the font, color, size, and spacing of your content.

/* Three ways to add CSS to HTML */

/* 1. Inline CSS */
<h1 style="color: blue;">Heading</h1>

/* 2. Internal CSS */
<style>
    h1 {
        color: blue;
    }
</style>

/* 3. External CSS */
<link rel="stylesheet" href="styles.css">

27. CSS Syntax and Selectors

CSS consists of style rules that are interpreted by the browser and then applied to the corresponding elements in your document.

/* CSS Syntax: selector { property: value; } */

/* Element Selector */
p {
    color: red;
    font-size: 16px;
}

/* Class Selector */
.highlight {
    background-color: yellow;
}

/* ID Selector */
#header {
    background-color: #333;
}

/* Universal Selector */
* {
    margin: 0;
    padding: 0;
}

28. CSS Colors and Backgrounds

CSS provides various ways to specify colors and control background properties of elements.

/* Color Values in CSS */

.color-examples {
    /* Named colors */
    color: red;
    background-color: blue;
    
    /* HEX colors */
    color: #ff0000;
    background-color: #0000ff;
    
    /* RGB colors */
    color: rgb(255, 0, 0);
    background-color: rgba(0, 0, 255, 0.5);
    
    /* HSL colors */
    color: hsl(0, 100%, 50%);
}

.background-examples {
    background-image: url('image.jpg');
    background-repeat: no-repeat;
    background-position: center;
    background-size: cover;
}

29. CSS Typography

CSS provides extensive control over text appearance, including font family, size, weight, style, and spacing.

/* Typography Properties */

.typography {
    /* Font family */
    font-family: 'Arial', 'Helvetica', sans-serif;
    
    /* Font size */
    font-size: 16px;
    font-size: 1rem;
    font-size: 100%;
    
    /* Font weight */
    font-weight: normal;
    font-weight: bold;
    font-weight: 700;
    
    /* Text alignment */
    text-align: left;
    text-align: center;
    text-align: justify;
    
    /* Text decoration */
    text-decoration: none;
    text-decoration: underline;
    
    /* Line height */
    line-height: 1.5;
    
    /* Letter spacing */
    letter-spacing: 1px;
}

30. CSS Box Model

The CSS box model is essentially a box that wraps around every HTML element. It consists of: content, padding, border, and margin.

/* CSS Box Model Properties */

.box {
    /* Content dimensions */
    width: 300px;
    height: 200px;
    
    /* Padding - space inside the border */
    padding: 20px;
    padding-top: 10px;
    padding-right: 15px;
    padding-bottom: 10px;
    padding-left: 15px;
    
    /* Border - around padding and content */
    border: 2px solid #333;
    border-radius: 8px;
    
    /* Margin - space outside the border */
    margin: 20px;
    margin-top: 10px;
    margin-bottom: 10px;
    
    /* Box-sizing */
    box-sizing: border-box; /* includes padding and border in width/height */
}

IV. JavaScript (Chapters 51-80)

51. Introduction to JavaScript

JavaScript is a programming language that enables interactive web pages and is an essential part of web applications.

// JavaScript can be added to HTML in three ways:

// 1. Inline JavaScript
<button onclick="alert('Hello World!')">Click me</button>

// 2. Internal JavaScript
<script>
    alert('Hello World!');
</script>

// 3. External JavaScript
<script src="script.js"></script>

52. JavaScript Variables and Data Types

Variables are containers for storing data values. JavaScript has dynamic types and several primitive data types.

// Variable Declaration
var oldVariable = "I'm function scoped";  // Avoid using
let changeableVariable = "I can be changed";
const constantVariable = "I cannot be reassigned";

// JavaScript Data Types
const stringType = "Hello World";
const numberType = 42;
const booleanType = true;
const nullType = null;
const undefinedType = undefined;
const objectType = {name: "John", age: 30};
const arrayType = [1, 2, 3];
const functionType = function() { return "I'm a function"; };

53. JavaScript Operators

JavaScript operators are used to assign values, compare values, perform arithmetic operations, and more.

// Arithmetic Operators
let sum = 10 + 5;      // 15
let difference = 10 - 5; // 5
let product = 10 * 5;   // 50
let quotient = 10 / 5;  // 2
let remainder = 10 % 3; // 1

// Comparison Operators
let isEqual = 10 == "10";    // true (type coercion)
let isStrictEqual = 10 === "10"; // false (no type coercion)
let isNotEqual = 10 != 5;     // true
let isGreater = 10 > 5;       // true

// Logical Operators
let andResult = true && false; // false
let orResult = true || false;  // true
let notResult = !true;         // false

54. JavaScript Control Structures

Control structures determine the flow of execution in a program through conditional statements and loops.

// Conditional Statements

// if-else statement
let age = 18;
if (age >= 18) {
    console.log("You are an adult");
} else {
    console.log("You are a minor");
}

// switch statement
let day = "Monday";
switch (day) {
    case "Monday":
        console.log("Start of work week");
        break;
    case "Friday":
        console.log("Weekend is coming!");
        break;
    default:
        console.log("It's a regular day");
}

// Loops
// for loop
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// while loop
let count = 0;
while (count < 5) {
    console.log(count);
    count++;
}

55. JavaScript Functions

Functions are one of the fundamental building blocks in JavaScript. A function is a set of statements that performs a task or calculates a value.

// Function Declaration
function greet(name) {
    return "Hello, " + name + "!";
}

// Function Expression
const greet = function(name) {
    return "Hello, " + name + "!";
};

// Arrow Function (ES6)
const greet = (name) => {
    return "Hello, " + name + "!";
};

// Arrow Function with implicit return
const greet = name => "Hello, " + name + "!";

// Function with default parameters
function createUser(name, age = 18) {
    return {
        name: name,
        age: age
    };
}

// Calling functions
const message = greet("Alice");
console.log(message); // "Hello, Alice!"

IX. Advanced Topics (Chapters 201-250)

201. Introduction to Advanced Topics

Advanced web development topics include performance optimization, security, accessibility, and modern web technologies.

// Advanced JavaScript: Promises and Async/Await

// Using Promises
function fetchData() {
    return new Promise((resolve, reject) => {
        setTimeout(() => {
            resolve("Data fetched successfully!");
        }, 2000);
    });
}

// Using Async/Await
async function getData() {
    try {
        const result = await fetchData();
        console.log(result);
    } catch (error) {
        console.log("Error:", error);
    }
}

getData();
